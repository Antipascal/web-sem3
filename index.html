What is this site for?

This is the site for tracking metrics and alerts

Metrics

Metrics are the main part of the monitoring system. They are used to measure the state of the system.

Alerts

Alerts are used to notify the user about the state of the system.

Why prometheus

Prometheus is a leading open-source monitoring solution

It provides powerful querying mechanism for selecting and aggregating time series data in human-readable format like this:
http_requests_total{environment=~"staging|testing|development",method!="GET"}

Benefits of self-made solution

  • Ability to create something that already exists, but make it worse
